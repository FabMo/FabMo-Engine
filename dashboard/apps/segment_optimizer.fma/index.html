Segment Optimizer

Combine runs of very small toolpath segments in an OpenSBP file so they execute efficiently. Cut moves (M2/M3/MX/MY/MZ) with literal coordinates are merged; everything else — arcs, jogs, speeds, spindle, pauses, tool changes, comments and any move using a variable — is left untouched.
