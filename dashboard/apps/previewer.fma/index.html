    Operations Panel

    Files compiled from SBP or with VCarve/ShopBot operation comments expose each operation as a checkable row. Use the panel to override tool defaults, run only what you need, or submit a trimmed/selected job.

    Setup Sets the file-wide defaults: Material thickness, Z origin (Material Surface or Table Surface), and the Default tool (Flat / Ball / V-Bit, plus diameter and V-bit angle). Changes here re-render the material simulation.
    Per-operation tool Each operation row has its own bit type, diameter, and (for V-bit) angle. Editing these overrides the Setup default for that operation only.
    Run Full Job Submits the entire file as loaded.
    Run Trimmed / Submit Trimmed Run or submit only the range between Cut Start and Cut Stop (set with I / O or the bottom-bar buttons).
    Run Selected / Submit Selected Run or submit only the operations whose checkboxes are ticked. The All link toggles every operation at once.

    AR & Top Overlays

    When a machine camera is detected and table corners have been calibrated, two extra toggles appear on the bottom bar:

    AR Overlays the toolpath preview on the live camera feed using the calibrated camera pose, so the rendered path appears in the same perspective as the real table.
    Top Un-warped top-down view of the live table. The camera image is warped to a true overhead, with the toolpath rendered over it — useful for spotting clamps or fixtures the path would collide with.
    ⟳ Recalibrate Re-marks the four table corners against the live video. Drag each handle (TL, TR, BR, BL) onto the corresponding corner of the table, then Save. Calibration is stored in the machine config and shared between AR and Top modes.

    Settings

    Simulation Speed Playback speed relative to real time (1/16x – 16x).
    Temp Change Boundary Units Temporarily changes units (inches vs mm) for the file boundary indicators. A unit setting inside the file overrides the machine setting for motion, but does not change the machine’s display units.
    Grid Step Grid step size in the current machine units.
    Table Offset from Z-Zero How far the table sits below the Z-zero plane. If you zero at the table, set 0. If you zero on top of 0.75" stock, set −0.75.
    Live Sync Buffer Compensation How many planner-buffered commands to skip ahead in the preview so the on-screen position tracks the actual machine more closely (SBP: SBP commands; GCode: gcode lines).
    Display checkboxes Show or hide grid, machine table, dimensions, axes, tool cone, and point-cloud / wireframe overlays.
    Reset Material Clears any cut material from the simulation surface and re-initializes a fresh stock block.
    Point Cloud Opacity Transparency of the leveler point-cloud overlay.
    Material Opacity Transparency of the simulated stock block.
    Light Azimuth / Elevation Direction of the scene light. Adjust for better depth shading on the cut surface.
    Simulation Resolution Height-map grid resolution for the material simulation. Higher is more accurate but slower to compute and re-render.
