AXIS CONFIGURATIONS i This page contains default/current settings for Axis characteristics:

Feedrate Maximum: The maximum allowable feedrate for this machine in a move
    speed command ([M] commands in OpenSBP or G1 commands in g-code).
Jog Velocity: The speed of Jog or rapid-traverse comands ([J]og commands
   in OpenSBP; G0 command in g-code). May be greater than Feedrate Maximum.
Maximum Jerk: "Jerk" is technically the 3rd derivative of position -- it is the
   rate of change in acceleration/deceleration and determines how quickly your tool changes
   from one speed/direction to another. A higher value here creates more aggressive motion.
   [recommended settings: in inches 50 - 250; in mm 1270 - 6350 ((units/ min³) * 1,000,000)]
   FabMo calculates a smooth, S-shaped speed curve that limits jerk to this setting.
High Brake Jerk: The "jerk" setting creating an abrupt stop for special
   situations such as homing and unprogrammed STOPS. If this value is too
   high or loads heavy, position accuracy may be lost.
Axis Mode: The movement mode for this axis. Linear axes are controlled in the
   default linear unit system. Rotary axes are controlled in degrees of rotation.
   Note that changing the Axis Mode will reset the default parameters for the axis.

Channel 1 i These settings provide the lowest level assignment for
each motor output driver channel. The driver channel is
assigned an Axis name, a direction, and a Unit Value.
The Unit Value is the number of steps (controller pulses)
per unit of distance (inches, mm, or degrees). The value
is responsible for setting how far the tool travels per
requested move. If a tool is consistently travelling a
proportionally incorrect distance, the Unit Value is
likely incorrect.

For ShopBot's Desktop tools, the correct Unit Value is:
    4000 steps/inch | 157.48031 steps/mm

and homing sensors are usually placed slightly
outside these limits.
