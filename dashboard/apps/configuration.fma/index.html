Channel 1 i These settings provide the lowest level assignment for
each motor output driver channel. The driver channel is
assigned an Axis name, a direction, and a Unit Value.
The Unit Value is the number of steps (controller pulses)
per unit of distance (inches, mm, or degrees). The value
is responsible for setting how far the tool travels per
requested move. If a tool is consistently travelling a
proportionally incorrect distance, the Unit Value is
likely incorrect.
For ShopBot's Desktop tools, the correct Unit Value is:
    4000 steps/inch | 157.48031 steps/mm
